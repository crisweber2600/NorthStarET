## Usage

All developers across all layers (Foundation, DigitalInk, future) should follow Spec-Kit Coach workflows:
- **Feature specification**: Use `/speckit.specify` to generate specs
- **Technical planning**: Use `/speckit.plan` to create implementation plans
- **Task breakdown**: Use `/speckit.tasks` to generate phased task lists
- **Constitution compliance**: Validate work against constitution rules
