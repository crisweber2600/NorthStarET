# Microservices Documentation

This directory contains documentation for the microservices in the NorthStarPlan system, organized by layer.

## Structure

*   **foundation/**: Documentation for Foundation Layer services (Identity, Gateway, Config, etc.).
*   **digital-ink/**: Documentation for Digital Ink Layer services (Student, Staff, Assessment, etc.).

## Principles

Refer to the [Constitution](../.specify/memory/constitution.md) for the core principles governing these services.
